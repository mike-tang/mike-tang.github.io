Demandforce Learn

Offline-enabled resource for quick product overviews

Overview

The main purpose of Demandforce Learn was to serve as a quick product overview for sales representatives and partners to use while traveling and speaking with potential customers. Since internet connectivity isn't always guaranteed at tradeshows or at various small business locations, DF reps needed an offline alternative to our website.

My Role

Interaction design, front end development, responsive web design

The Solution

To solve this problem, I designed and built a lightweight HTML5 webpage with smooth CSS3 transitions.

The wireframes were relatively simple, so I skipped drafting mockups and went straight into developing and iterating on a prototype that would eventually launch as the live page.

Without a doubt, this webpage needed to be fluid responsive so that it could cater to all types of mobile devices. I built a custom grid that responded to landscape and portrait views on tablets and phones, and used FitVids.js to make sure videos resized properly as well. To further improve performance and flexibility, I built an icon font with IcoMoon. Finally, I enabled offline access by listing all assets in the cache manifest, which tells the browser what to store in the application cache.
